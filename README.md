## Installation

1. Download the DMG file
2. Open it and drag ClaudeBench to Applications
3. Launch and follow the setup wizard to enter your API key

## System Requirements

- macOS 13.0 (Ventura) or later
- Apple Silicon (M1/M2/M3/M4) or Intel Core i5
- 500 MB available storage
