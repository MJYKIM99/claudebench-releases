# Security Policy

This repository is a public release distribution repository for ClaudeBench.

## Scope

Use this repository for reporting security issues related to:
- release asset integrity
- tampered or incorrect download artifacts
- checksum mismatches
- public release metadata that could mislead users

Do not disclose sensitive security issues publicly in GitHub issues.

## Reporting

Send security reports through the support channel on:

- https://claudebench.com/support

Include:
- affected version
- asset URL
- observed behavior
- reproduction details
- any integrity verification output

## Notes

- Public GitHub issues are not the right place for security disclosures.
- This repository does not contain the private application source code.
