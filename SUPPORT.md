# Support

For product support, installation help, or release download issues, use:

- Website: https://claudebench.com
- Download page: https://claudebench.com/download
- Support: https://claudebench.com/support

## What This Repository Is For

This repository is for public release assets and release notes.

Appropriate issues here include:
- broken release links
- incorrect release metadata
- checksum problems
- install or notarization issues with published builds

## What Does Not Belong Here

- private source code requests
- internal roadmap questions
- account or billing details posted publicly
- secrets, keys, or private build information
